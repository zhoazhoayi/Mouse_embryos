本研究使用的胚胎图像数据集为专门构建用于胚胎发育监测的私有时序数据集，专注于记录胚胎的动态发育过程。该数据集共包含594张图像，每张图分辨率为5120×5120，采集时间跨度为2025年6月14日至2025年8月30日，共涵盖三个独立的培养批次，每次胚胎培养时间不等，胚胎图像每天采集一次，每次每个栅格采集9张不同焦面图像，图像采集情况如表1所示，采集图像如图5所示。为了确保模型评估的严谨性与泛化能力的可靠性，本研究将原始数据集按7:2:1比例随机划分为训练集、验证集与测试集。因为每九张图像为一组，所以实际数据集中图像数量比例为423:117:54。针对本研究的目标检测任务，所有原始图像均使用 LabelMe 工具进行人工标注。
