The embryo image dataset used in this study is a private time-lapse dataset focused on recording the dynamic developmental process of embryos. The dataset comprises a total of 594 images, each with a resolution of 5120×5120. The images were acquired between June 14, 2025, and August 30, 2025, covering three independent culture batches. The duration of embryo culture varied across batches. Embryo images were acquired once daily, with nine images captured at different focal planes for each grid location during every acquisition, aiming to capture diverse samples under various culture conditions, The image acquisition details are summarized in Table 1, and the collected images are shown in Figure 5.To ensure the fairness and generalization capability of the model training and evaluation, the dataset was randomly divided into training, validation, and test sets using a 7:2:1 ratio. Since the images are grouped in sets of nine, the actual number of images in the dataset is split as 423:117:54. For the object detection task of this study, all raw images were manually annotated using the LabelMe tool.
